Политика конфиденциальности

1. Общие положения

Сайт "Гармония Тела" является учебным проектом и не собирает, не хранит и не обрабатывает персональные данные пользователей. Все расчеты (калории) выполняются локально в вашем браузере.

2. Локальное хранение данных

На сайте используется только:

  • LocalStorage - для сохранения темы оформления
  • Session - временные данные сессии

Эти данные не передаются на сервер и удаляются при очистке кэша браузера.

3. Файлы cookie

Сайт не использует файлы cookie для отслеживания. Все функциональные cookie являются техническими.

4. Контактная информация

Поскольку это учебный проект, официальной контактной информации не предусмотрено.
